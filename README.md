## Overview
A modern news platform showcasing the latest in Agentic AI and Enterprise solutions.

![Preview](./preview.png)

## Architecture
- **Web App**: Next.js
- **API**: FastAPI
- **Storage**: PostgreSQL & Milvus

## Getting Started
1. Install Docker & Docker Compose.
2. Run `docker-compose up -d`.
3. Open `http://localhost:3000`.
